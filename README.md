# Drive Direct 26 🚗

Drive Direct 26 is a modern automotive marketplace platform designed for dealers and private sellers to list vehicles, manage customer inquiries, and seamlessly monetize through featured listings, premium consultations, and integrated advertising.

---

## 🚀 Tech Stack

The platform is engineered using a robust, production-ready stack:

*   **Framework:** Next.js (App Router)
*   **Database ORM:** Prisma
*   **Database Provider:** Supabase (PostgreSQL)
*   **Authentication:** Clerk
*   **Content Management:** Sanity CMS
*   **Media Hosting:** Cloudinary
*   **Email & Marketing:** Brevo (formerly Sendinblue)
*   **Payment Gateway:** PayFast

---

## ✨ Core Features

*   **Vehicle Listings:** Effortless creation and management of car listings for both private sellers and commercial dealerships.
*   **Monetisation Engine:** Built-in support for featured premium listings, paid expert consultations, and display advertisements.
*   **Inquiry Management:** Seamless communication layer connecting buyers directly to sellers.
*   **Media Optimization:** Lightning-fast image transformations and storage powered by Cloudinary.
